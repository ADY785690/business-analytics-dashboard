# 📈 Stock Market Intelligence Dashboard

A professional Stock Market Intelligence Dashboard built using Python, Streamlit, Yahoo Finance and Data Analytics techniques.

## 🚀 Live Features

### 📊 Real-Time Stock Analysis
- Live stock data using Yahoo Finance API
- Historical stock performance tracking
- Interactive stock selection

### 📈 Technical Analysis
- Closing Price Trend Analysis
- 50-Day Moving Average Analysis
- Volatility Analysis
- Return Analysis

### 💰 Portfolio Simulation
- Investment amount simulation
- Share holding estimation
- Portfolio value calculation

### 📉 Risk Assessment
- Annualized Volatility Calculation
- Risk Monitoring Dashboard
- Performance Evaluation

### 📋 Data Visualization
- Interactive Charts
- KPI Dashboard
- Historical Data Tables
